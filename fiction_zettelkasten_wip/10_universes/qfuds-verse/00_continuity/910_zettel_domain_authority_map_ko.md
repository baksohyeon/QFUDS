# 캐논 권위는 도메인별로 갈린다

```text
fiction/provenance only
research evidence: no
canon_state: authority summary from 000
```

## Statement

QFUDS Verse의 canon은 하나의 거대 문서가 아니라 복원, 명칭, 인과, 연표, 시간 척추,
인물, 주제축 같은 도메인별 권위 문서로 관리된다.

## Source Refs

- [000 캐논 권위·SSOT 지도 §도메인별 권위 문서](000_canon_authority_and_ssot_map_ko.md#도메인별-권위-문서-충돌-시-이-문서가-최종): 충돌 시 어느 문서가 이기는지 도메인별로 적는다.
- [000 §전역 충돌 해소 규칙](000_canon_authority_and_ssot_map_ko.md#전역-충돌-해소-규칙): 명시적 우선 선언, 권위 표, last_updated, unknown 순서를 둔다.

## Context

제텔카스텐 카드는 빠르게 연결되기 때문에, 출처 권위를 흐리기 쉽다. 이 카드는 카드
간 링크보다 도메인 권위가 우선한다는 운영 규칙을 붙잡는다. 좋은 연결이 canon 충돌을
자동 해결하지 않는다.
