# <Work / Arc> Continuity Audit

Run before promoting a draft toward release. Cross-checks a scene or episode
against the work's continuity reference (a work's "Work Bible" section in
`fiction/projects/<work-id>/README.md`, or the universe's
`fiction/worlds/<universe-id>/series-bible/`) so revisions do not silently
break canon. Supports the
[Fiction IP Management Workflow](../../workflows/fiction-ip-management-workflow.md).

## Multi-Version Continuity (only if multiple versions exist)

- [ ] Story state, events, outcomes, and order match the declared adaptation policy.
- [ ] Proper nouns and deliberate differences follow the work's style packet.
- [ ] Each target-language version reads naturally in its own register.

## Meaning Preservation (revision safety)

- [ ] Facts, numbers, quotations, causal links unchanged by the revision.
- [ ] No new external-source / product claim introduced.
- [ ] Plot order intact.
