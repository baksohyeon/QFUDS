# QFUDS v0.1 Theory Note

Date reconstructed: 2026-06-08

## Status

v0.1 is the conceptual origin stage. It is preserved to explain why the idea was introduced, not because it is a validated theory.

## Assumptions

1. Information has physical cost, motivated by Landauer-style reasoning.
2. Black holes are natural places where information, entropy, quantum mechanics, and gravity meet.
3. Hawking radiation may preserve information in an extremely scrambled form in a unitary description.
4. A reverse-channel or remnant question is worth asking, but not assumed to be established physics.
5. Quantum spacetime foam may provide an effective medium with long-wavelength collective behavior.
6. The observed dark sector might be modeled as two large-scale modes of that medium.

## Equations

v0.1 had no validated equations. The only useful schematic split was:

```text
dark sector = clustering foam-like mode + smooth residual-pressure mode
```

with target behavior:

```text
clustering mode:      w ~= 0,  c_s^2 ~= 0
residual mode:        w ~= -1
optional remnant:     subdominant compact or defect-like component
```

## Physical Interpretation

The original idea linked several questions:

```text
information cost
-> black-hole information preservation
-> possible reverse or delayed-return channel
-> quantum foam as information-bearing medium
-> dark matter as clustering foam mode
-> dark energy as smooth residual pressure
```

The white-hole-universe image belongs to this origin stage only. It is not the maintained central claim.

## Differences From Previous Version

There is no earlier model version in this repository.

## Unresolved Issues

1. No microscopic action.
2. No stress tensor.
3. No background equations.
4. No perturbation equations.
5. No observational prediction beyond broad analogy.
6. No reason yet for the clustering phase to have near-zero sound speed.
7. No remnant abundance, mass spectrum, or coupling model.

## Decision

Preserve v0.1 as research genealogy. Do not use it as evidence for viability.
