# 02 Theory

Theory and literature-positioning documents.

This folder is not the experiment record. It defines hypotheses, assumptions, equations, and model versions.

Use these folders for validation evidence:

- `docs/03_experiments/`: experiment definitions and reproducible commands.
- `docs/04_results/`: result reports, analysis reports, figures, and decisions.
- `outputs/`: generated CSV and PNG artifacts from the numerical runs.

- `qfuds_v0_1.md`: initial conceptual theory note.
- `qfuds_v0_2.md`: minimal two-phase effective-fluid note.
- `qfuds_v0_3.md`: physically labeled `Gamma(a)` theory note.
- `qfuds_research_report.md`: adversarial literature comparison and mathematical formulation.
