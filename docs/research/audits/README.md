# Availability Audits

This folder stores dated checks for paper data products.

Availability audits may record:

- search queries;
- source pages checked;
- public tables, chains, posterior samples, covariance matrices, code, and
  figures found;
- products not found;
- whether author data or figure digitization is likely required.

Availability audits must not record QFUDS conclusions. A cached "not found"
entry means only that the product was not found in the checked sources on the
recorded date.
