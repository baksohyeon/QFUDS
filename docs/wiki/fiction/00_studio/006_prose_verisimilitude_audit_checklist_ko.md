# Fiction Prose 핍진성·개연성 장면 감사 체크리스트

## 목적

장면 단위로 핍진성(있을 법함)과 개연성(앞뒤가 맞음)을 확인하는 재사용
체크리스트. SAGA 세계 기준점 문서의 산문형 5규칙을 "장면마다 예/아니오로
답하는" 항목으로 승격한다.

원천 규칙: [SAGA 세계 기준점과 핍진성 규칙](../10_universes/qfuds-verse/10_world/001_world_anchor_and_verisimilitude_ko.md)의
핍진성 5규칙. craft 일반 틀은
[Creative Writing Craft Harness](004_creative_writing_craft_harness_ko.md).

이 문서는 reference다. QFUDS 연구 증거가 아니다. 새 외부 source claim 없음.

## 사용법

장면(scene) 하나를 고른다. 아래 5블록을 순서대로 묻는다. "아니오"가 나오면
그 장면은 설명을 추가하지 말고 **절차·비용·계급·일상**으로 고쳐 쓴다(설교 금지).

## 1. 기적에는 기관이 있는가

- [ ] 이 장면의 기술/기적을 **누가 소유·운영**하는가가 보이는가?
- [ ] 비용을 **누가 지불**하고 누가 **audit**하는가?
- [ ] 누가 **배제·두려워**하는가?
- 실패 신호: 기술이 공중에 뜬 wonder로만 존재. → 기관 하나를 장면에 박는다.

## 2. 기술은 계급을 만드는가

- [ ] access 불평등이 장면에 **구체물**로 드러나는가? (요금표, 등급, 색깔 표식)
- [ ] 고해상/저해상, 권리귀환/증언귀환/데이터귀환의 차이가 인물 처지로 보이는가?
- 실패 신호: 모두가 같은 조건. → 한 인물의 손해를 계급으로 환산한다.

## 3. 철학이 절차가 되었는가

- [ ] 추상 질문이 **양식·심리·의식·가격표** 같은 절차로 처리되는가?
- [ ] 인물이 철학을 강의하지 않고 절차 안에서 **행동/저항**하는가?
- 실패 신호: 인물이 주제를 대신 설명. → 같은 뜻을 절차적 동작으로 바꾼다.

## 4. 현재 reference가 화석이 되었는가

- [ ] Bitcoin·AI 밈·초기 LLM 실패 등 현재물이 **유물 형태**(archive card,
  museum plate, court exhibit, dead metaphor)로 나오는가?
- [ ] 원래 의미가 **decay된 뒤**의 감각인가? (2026년이 우주선만 단 채 그대로 오지 않음)
- 실패 신호: 현재 정치/투자 논쟁 톤. → 후대의 거리감을 입힌다.

## 5. 평범한 일상이 있는가

- [ ] 거대 설정 옆에 **사소한 루틴**(요금, 줄, 식은 차, 젖은 신발)이 있는가?
- [ ] 인물이 기적을 지루해하거나 행정처럼 다루는 순간이 있는가?
- 실패 신호: 모든 문장이 장엄. → 평범한 동작 하나로 호흡을 떨어뜨린다.

## 개연성(인과) 추가 점검

- [ ] 사건 A→B 사이에 **건너뛴 인과**가 없는가? (편의적 전개 금지)
- [ ] 인물의 선택이 그가 **아는 정보·동기**로 설명되는가?
- [ ] 반전이 단서 없이 떨어지지 않는가? (앞에 심어진 단서가 있는가)

## 문체 점검 (허풍 방지)

- [ ] 한 단락에 은유가 과적되지 않았는가? (장식이 사건을 가리지 않음)
- [ ] 현학적 대사는 그 인물의 voice가 **정당화**할 때만 남겼는가?
- [ ] 추상명사 나열 대신 **구체 명사·동작**으로 말하는가?

## AI 티 금지 (단어·패턴 블랙리스트)

AI가 쓴 한국어 prose에서 반복되는 클리셰. 한 번 정도는 괜찮지만 **자주 나오면
어색**하다. 검수 시 빈도를 센다.

금지/주의 단어(분위기용 추상명사 남발):

```text
영토, 성역, 심연, 궤적, 잔향, 파편, 침묵, 껍질/껍데기,
균열, 지평, 요람, 등대, 나침반, 교향곡, 하모니
```

금지/주의 문장 패턴:

- [ ] `~의 ~다` 단정 구문을 남발하지 않았는가? (예: "그것은 침묵의 언어다")
- [ ] `~가 아니라 ~다`(not A but B) 패턴이 한 글에 몇 번 이상 반복되지 않는가?
- [ ] `단순히 ~을 넘어` 패턴을 쓰지 않았는가?
- [ ] `엔진:`, `핵심은 A가 아니라 B`, `사건의 주어는 X`, `한 사람의 손가락에 올린다`
  같은 기획서용 훅 포장문을 산문이나 기준 문장으로 남기지 않았는가?
- [ ] **본론에 비해 결론이 거창한** 태도(작은 장면 끝에 우주적 선언)를 붙이지
  않았는가?

전용 에이전트 `ai-tell-detector`로 한 번 더 스캔하고, 잔존 시
`korean-style-rewriter`/`humanize-korean`으로 수술적으로 고친다.
