# QFUDS Verse 세계 밀도 마스터 인덱스 (030~036 + 도메인 커버리지)

## 이 문서의 자리

`qfuds-verse` 세계 밀도가 어디에 얼마나 쌓였는지 **한 장에서 잡는 카탈로그 색인**이다.
[Fiction Catalog](README.md)의 canon/continuity 색인 역할이며, 현재 유일 활성 유니버스인
qfuds-verse의 세계 밀도 상태판이다.

웨이브 전용 크로스 인덱스(030~035)는
[035 capstone §4](../10_universes/qfuds-verse/10_world/035_world_expansion_wave6_ecology_education_media_index_ko.md)가
보유한다. 이 문서는 그 위에 (a) 심층시간 연대기
[036](../10_universes/qfuds-verse/00_continuity/036_far_future_deep_time_chronicle_ko.md)을 포함하고,
(b) 듄급 14도메인 커버리지 지도, (c) 구체 인벤토리, (d) 캐논/후보 상태를 더한다. 035를
대체하지 않는다. 035는 웨이브 시리즈의 capstone, 이 문서는 밀도 전체의 카탈로그 색인이다.

```text
fiction/provenance only

캐논 상태: 관리용 색인(candidate register). 새 캐논·새 고유명 생성 안 함
```

## D. 상태·거버넌스

- **캐논 아님:** 030~035 밀도는 전부 candidate, 전부 보류(사용자 2026-07-01). 원고(오르페우스·
  마라·캐스)가 이름을 당길 때
  [030 §7 6체크](../10_universes/qfuds-verse/10_world/030_world_expansion_wave1_names_places_events_ko.md)로
  개별 승격.
- **036만 예외:** continuity 연표 확장 레이어. 002 SSOT를 대체하지 않는다(그 위). 충돌 시
  021 > 002/011/025 > 036.
- **불변 가드:** 복원=사본 절대(021) / 필드마크 순서 불변(026 §4) / 명칭 SSOT(015) / 실존
  집단 비대응 / warlord·신체 호러 금지.
- **확장:** `10_world` 세계 확장의 다음 웨이브는 037+로 증분(035 명시). 이 카탈로그 색인은
  그 웨이브 번호를 소비하지 않는다(catalog 선반 소속).

## 요약

"이름 → 관계 → 공간 → 경제·의례 → 언어·기술 → 생태·교육"의 6층(030~035)에 시간축 연대기
(036)가 더해져 듄급 밀도 뼈대가 섰다. 다음 후보 액션: (1) 036을 002 골격에 정식 승격,
(2) 특정 원고용 이름 6체크 승격, (3) 웨이브 7(037+) 신규 도메인.
