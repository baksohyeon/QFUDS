# QFUDS SAGA 첫 Arc 6화 Outline

## 목적

이 문서는 QFUDS SAGA의 첫 arc를 여섯 episode로 나누는 개발 outline이다.

기준일은 2026-06-19다. 이 날짜는 author-side baseline이다. 작중 시간은
Continuity Court Era이며, Bitcoin/Genesis Chain, Cryptographic Death,
Mara Veyr, Hawking restoration, QFUDS는 모두 fiction/provenance 소재다.

목표는 첫 화에 모든 설정을 몰아넣지 않는 것이다.

```text
1화는 질문을 열고,
2-4화는 세계가 그 질문으로 먹고사는 방식을 보여 주고,
5-6화는 그 질문이 우주 규모의 금기로 확장되는 순간을 보여 준다.
```

## Episode 1: Exhibit S-0

### 기능

절차적으로는 이미 끝난 듯한 court case로 시작한 뒤, 그 결론을 도덕적으로
사용 불가능하게 만든다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| Court fragment | 직접 장면이 시작되기 전에 Mara가 identity threshold를 통과했음을 보여 준다 |
| Waiting City | restoration이 기적이 아니라 일상 infrastructure임을 보여 준다 |
| Laur office | Liora가 빠른 closure보다 `unknown`을 더 중시함을 보인다 |
| Genesis Chain mention | old key restoration이 person restoration과 법적으로 이웃해 있음을 심는다 |
| Mara entrance | 사건을 이론이 아니라 몸을 가진 사람의 문제로 만든다 |
| Refusal | Mara가 법정이 배정한 역할을 거절한다 |
| Archive correction | 질문을 identity에서 right-to-loss로 이동시킨다 |

### Character pressure

Liora는 좁은 procedural delay를 원한다. 다른 모두는 closure를 원한다. 그녀의
첫 승리는 작고 보기 좋지 않다. Mara를 법적으로 unresolved 상태로 묶어 두는
것이기 때문이다.

### Cut line candidate

```text
돌아온 사람이 누구인지 이름 붙이기 전에, 무엇이 복원됐는지 먼저 정의하라.
```

## Episode 2: The Dead Exchange

### 기능

Genesis Chain을 구체화한다. 강의 속 lore가 아니라, 죽은 돈이 산 claim을 아직도
결정하는 장소로 만든다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| Dead exchange basement | 가격은 없고 custody genealogy만 남은 공간 |
| Ledger House petition | 죽은 key가 살아 있는 succession claim에 쓰인다 |
| Liora audit | recovered access와 moral authority를 분리한다 |
| Aletheia warning | consent는 key처럼 상속될 수 없다고 경고한다 |
| contradiction | reconstructed key가 살아 있고 부인하는 heir를 지목한다 |

### Character pressure

Liora는 Mara의 사건이 특수한 예외가 아니라는 것을 보기 시작한다. 법 체계는
이미 property case에서 "recovered access"를 precedent로 받아들였다.

### Cut line candidate

```text
chain은 죽었다. 그 상속자들은 죽지 않았다.
```

## Episode 3: The Last Hodler

### 기능

사용자의 Bitcoin 불안을 현실 market claim으로 쓰지 않고 신화로 바꾼다. 핵심은
price가 zero가 되었는지가 아니라, price가 의미를 잃은 뒤 무엇이 살아남았는가다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| academy lecture | 학생들은 Last Hodler를 사람이 아니라 archetype으로 배운다 |
| public protest | 어떤 이들은 Genesis Chain을 freedom relic이라 부르고, 다른 이들은 first cage라 부른다 |
| Mara deposition | Mara가 key recovery에서 빌려온 언어를 거부한다 |
| Elias interview | grief가 restoration vocabulary로 말한다 |
| Archive classification | Bitcoin이 "civilization-scale consent artifact"가 된다 |

### Character pressure

Elias는 진심이기 때문에 더 위험해진다. 그는 Mara 위에 군림하고 싶은 것이
아니다. 그는 사랑이 상실보다 오래갈 수 있는 세계를 원한다. 바로 그 욕망 때문에
그녀를 침해할 수 있다.

### Cut line candidate

```text
그는 price에 대해 틀렸다.
그는 memory에 대해 맞았다.
```

## Episode 4: Identity Flood

### 기능

한 명의 restored person에서 너무 많은 plausible restoration으로 escalation한다.
그래야 Preimage Restoration이 깔끔한 proof machine으로 굳지 않는다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| Null-Key attack | 폭탄은 없고 plausible memory graph 수천 개만 있다 |
| court paralysis | procedure는 어느 compatible self가 중요한지 고르지 못한다 |
| Mara-compatible graph | 한 graph가 Elias가 결코 알지 못한 기억을 포함한다 |
| Liora choice | Liora는 그 graph를 sabotage로 버리기를 거부한다 |
| Archive silence | machine은 해결하지 않고 resolution을 보류한다 |

### Character pressure

Mara는 더 이상 Elias 한 사람에게서만 자신을 지키는 것이 아니다. system이
admissible하게 만들 수 있는 모든 자기 가능성에게서 자신을 지켜야 한다.

### Cut line candidate

```text
당신들은 나를 늘린 것이 아니다. 나를 소유하는 방법을 늘렸을 뿐이다.
```

## Episode 5: Hawking Court

### 기능

Genesis Chain이 진짜 문제의 작은 버전일 뿐이었다는 사실을 드러낸다.
Cryptographic Death는 끝이 아니었다. 그것은 cosmic reverse computation을 위한
훈련이었다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| sealed exhibit | black-hole restoration protocol이 taboo 아래 법정에 들어온다 |
| Laur objection | Liora는 court가 physics를 law로 수입하고 있다고 주장한다 |
| Ledger argument | key recovery가 admissible이라면 state recovery도 그래야 한다고 밀어붙인다 |
| Aletheia split | 일부는 justice를 위해 cosmic audit를 원하고, 일부는 final captivity라 부른다 |
| forbidden term | QFUDS가 rejected but tempting interpretive model로 등장한다 |

### Character pressure

Liora는 court가 Mara를 이용해 훨씬 큰 project를 합법화하려 한다는 것을 이해한다.
그 project는 죽은 자를 evidence가 되도록 강제하는 일이다.

### Cut line candidate

```text
black hole을 audit할 수 있다면, 어떤 무덤도 닫힌 것이 아니다.
```

## Episode 6: The Broken Crown

### 기능

아무것도 해결하지 못하고 다음 arc를 필요하게 만드는 판결로 첫 arc를 닫는다.

### 장면 spine

| Beat | 장면 기능 |
| --- | --- |
| city fracture | Ledger Houses, Aletheia cells, families, courts가 공개적으로 갈라진다 |
| Elias confession | 그는 Mara의 refusal을 무시한 것이 아니라 앞질러 달아나려 했다고 인정한다 |
| Mara claim | 그녀는 original이나 copy 판정을 원하지 않고 loss로 대우받기를 요구한다 |
| Liora ruling draft | Liora가 새 category를 만든다: recoverable but not claimable |
| Archive correction | Last Archive가 civilization scale에서 질문을 다시 쓴다 |
| final image | 법정이 빈 뒤에도 Broken Crown exhibit이 켜져 있다 |

### Character pressure

Liora는 procedural category를 얻고 세계를 잃는다. 그녀는 한 형태의 possession은
막지만, cosmic scale에서 다시 시도할 machinery를 드러낸다.

### Cut line candidate

```text
court는 Mara가 그의 아내가 되는 것을 막았다.
그녀가 쓸모 있어지는 것은 막지 못했다.
```
