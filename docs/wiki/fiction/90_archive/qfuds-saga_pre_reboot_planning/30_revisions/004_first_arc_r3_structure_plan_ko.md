# QFUDS SAGA 1부 R3 구조 패스 계획

## 왜 R3인가

6인 페르소나 재테스트에서 R2 효과는 확인됐다(하드 이탈 2건→1건, 2편 이탈 해소).
남은 만장일치 병목은 문장이 아니라 **구조**다.

1. **표식 의식 반복.** 매 편 `검은 상자 → 표식 목록 → Liora가 손으로 씀 → Last
   Archive가 질문 고침 + STATUS 대구`가 동일. 3편째 예측됨(테드창="격언 공장").
   ep5가 물리로 변주를 주자 9점 → 변주가 답.
2. **QFUDS 공란.** 표제 이론인데 실체가 없다(테드창). 한 줄 실체 필요.
3. **ep5 물리 강의 집중.** 장르 독자는 견디나 캐주얼·웹소설 미끄러짐.
4. 부수: ep4 시각화, 격언 추가 컷, ep1 비트코인 압축(ep6 회수 보호), 영어 밀도.

## 불변 원칙 (사수)

- **필드마크 사슬은 유지한다.** RECOVERABLE / NOT CLAIMABLE → ACCESS != AUTHORITY →
  NO CONSENT BY ANALOGY → PLURALITY IS NOT CONSENT → PHYSICS IS NOT JURISDICTION →
  `who may author loss`는 throughline(SF애호가 호평). **마크 자체는 지운다고
  하지 않는다.** 바꾸는 것은 **그 마크에 도달·전달하는 의식**이다.
- 강점 사수: 도입 3연타, Mara/Noor, 편지, ep6 착지, 비트코인 챕터.
- 철칙: em dash 0(한·영), 표식=mark, 민감 주제 0, 플롯·수치·고유명사 불변.

## 변주 레버 (편마다 다르게 고른다)

같은 마크라도 닫는 방식을 달리한다.

- 다른 인물이 마크를 끌어낸다(Liora 손이 아니라 Noor/Ione/Mara의 행위·말로).
- 마크가 승리가 아니라 **상실·패배·대가**로 떨어진다.
- Last Archive 개입을 **생략**하거나, 질문 교정 대신 다른 형식으로.
- STATUS 대구(`A / B`) 출력을 **줄이거나 비대칭**으로.
- 장면 종류를 바꾼다(법정 일변도 탈피: 거리·사적 공간·이동·침묵).

## QFUDS 실체 한 줄 (테드창 공란 보강)

ep5에서 QFUDS를 "폐기된 셈법"으로만 두지 말고, 한 줄로 무엇을 주장하는 이론인지
명시한다. 예: 우주가 정보를 잃지 않고 어딘가에 흘려 둔다면, 충분히 모으면 잃은
상태를 되짚을 수 있다고 본 이론. 단 검증·재현이 안 돼 학계가 폐기했다는 in-world
지위는 유지(과장·정설화 금지).
