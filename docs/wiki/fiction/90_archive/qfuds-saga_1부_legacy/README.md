## 왜 _versions가 아니라 여기인가

읽기 가능한 직전 prototype과 v3 스냅샷(현행 029 계보의 과거)은 시리즈 안
`20_drafts/2부/_versions/`에 둔다. 이 폴더는 그보다 더 오래된 **폐기 1세대**라,
시리즈 밖 전역 아카이브(`90_archive/`)로 분리했다. 내부 상호 링크는 보관 시점
기준의 동결 상태이며 갱신하지 않는다.
