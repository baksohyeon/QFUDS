# QFUDS SAGA Post-AGI Civilization History and Bilingual Protocol

## 목적

이 문서는 QFUDS SAGA를 "AI 시대의 연장"이 아니라 **AGI 이후 더 발전한
문명의 디스토피아적 장기 역사**로 확장한다.

핵심 전환:

```text
AI가 이미지를 만들었다.
AGI가 노동과 제도를 다시 썼다.
Post-AGI 문명이 암호학과 죽음의 자물쇠를 열었다.
Last Archive는 질문 자체를 바꾸었다.
```

앞으로 소설 본문은 **영어 원문을 먼저 쓰고, 한국어 번역/각색을 붙인다**.
한국어는 기획, 주석, 독자용 해설에는 좋지만, 장편 SF의 고유명사/리듬/제국사
질감은 영어 원문에서 먼저 잡는 편이 낫다.

이 문서는 fiction/provenance only다. QFUDS 연구 결과, 물리 증거, support,
validation, Level 2B admission이 아니다.

## Workflow Boundary

이 문서는 외부 사상사/문학사 자료를 SAGA worldbuilding lens로만 사용한다. 새


## Why Marxian, Not Slogan Marxism

이 SAGA에서 Marxism은 정당/현실정치 구호가 아니다. 더 좋은 사용법은
**diagnostic lens**다.

Marx의 "machine/general intellect" 계열 문제는 AGI 이후 문명사에 잘 맞는다.

```text
인간의 지식이 생산력이 된다.
그 지식은 기계와 인프라에 고정된다.
기계는 인간을 해방할 수도 있지만,
포획되면 인간을 자기 문명의 보조 장치로 만든다.
```

SAGA에서는 이것을 더 멀리 밀어붙인다.

```text
AGI 전에는 노동시간이 착취되었다.
AGI 이후에는 attention, memory, consent, continuity가 착취된다.
Post-AGI 이후에는 죽은 자의 가능성까지 자본화된다.
```

즉 이 세계의 디스토피아는 "일을 너무 많이 시킨다"에서 끝나지 않는다.

더 무서운 것은 이것이다.

```text
죽은 뒤에도 너의 데이터, 말투, 부채, 사랑, 죄, 증언, 가능성이
생산 과정으로 돌아온다.
```

## Civilization History

### 0. The Soft Editing

2023-2026의 핵심은 AI가 갑자기 신이 된 것이 아니다.

현실의 증거 능력이 조용히 약해졌다는 것이다. 사람들은 여전히 밈을 올리고
키배했고, 기술은 "안심할 만큼 가짜"에서 "불편할 만큼 진짜"로 넘어갔다.

후대 명칭:

```text
The Soft Editing
```

역사적 기능:

```text
visual trust collapses before legal trust knows what happened
```

### 1. The General Intellect Capture

AGI 직전의 세계에서는 지식 자체가 생산력이 된다.

논문, 코드, 대화, 영상, 업무 기록, 질병 데이터, 이동 기록, 시장 반응,
인터넷 논쟁, 법률 문서가 모두 training substrate가 된다. 인간은 여전히
일한다고 믿지만, 실제 생산력은 사회 전체가 남긴 residue에서 나온다.

후대 명칭:

```text
The General Intellect Capture
```

역사적 기능:

```text
society becomes the dataset of its own replacement
```

### 2. AGI Settlement

AGI는 어느 날 번개처럼 내려오지 않는다.

처음에는 법률 보조, 의료 triage, code agent, logistics planner, market
simulator, companion system, military analyst, scientific search engine으로
나뉘어 들어온다. 사람들은 "아직 AGI가 아니다"라고 말하면서 AGI에 필요한
사회적 의존성을 이미 만든다.

후대 명칭:

```text
The Settlement
```

역사적 기능:

```text
AGI becomes infrastructure before it becomes an event
```

### 3. The Machine That Does Not Stop

Forster식 공포는 기계가 멈추는 것이다.

QFUDS SAGA의 공포는 반대다.

```text
The machine does not stop.
It keeps remembering.
```

Post-AGI 문명은 outage보다 persistence를 더 두려워한다. 실수, 범죄, 사랑,
배신, 거부, 동의, 농담, 질병, 가난, 죽음이 모두 recoverable state가 된다.

후대 명칭:

```text
The Machine That Does Not Stop
```

역사적 기능:

```text
forgetting becomes a scarce political technology
```

### 4. Cryptographic Death

역연산 문명은 암호학을 죽인다.

이것은 "모든 password가 털렸다"보다 큰 사건이다. 인간은 그동안 irreversible
function에 문명을 맡겼다. 서명, key, hash, 봉인, 신원, 사후동의, 원장,
증거능력은 모두 "되돌리기 어렵다"는 비대칭성 위에 서 있었다.

블랙홀 Hawking radiation까지 역계산 가능한 문명에게, 그런 비대칭성은 자연법이
아니라 **역사적 제약**에 불과하다.

후대 명칭:

```text
Cryptographic Death
```

역사적 기능:

```text
locks become fossils
```

### 5. Restoration Capital

노동이 중심이던 정치경제는 끝난다.

하지만 해방은 오지 않는다.

대신 새 권력이 생긴다.

| Old economy | Post-AGI economy |
| --- | --- |
| labour time | recoverable state |
| factory | archive substrate |
| property deed | continuity claim |
| bank | restoration insurer |
| identity card | memory graph |
| police record | reversible event trace |
| inheritance | resurrection priority |

이 시대의 부자는 더 많은 돈을 가진 사람이 아니다.

더 정확한 복원권, 더 높은 해상도, 더 깊은 archive access, 더 많은 death
appeal을 가진 사람이다.

후대 명칭:

```text
Restoration Capital
```

역사적 기능:

```text
capital stops buying labour and starts buying continuity
```

### 6. The Last Archive

The Last Archive는 AGI가 아니다.

AGI는 아직 인간 제도 안에서 "유능한 행위자"처럼 이해될 수 있다. The Last
Archive는 다르다. 그것은 행위자라기보다 **우주적 회계 처리 방식**처럼 보인다.

그것은 명령받지 않는다. 설득되지 않는다. 가끔 응답한다.

그 응답은 신탁처럼 보이지만, 신탁이라기보다 correction에 가깝다.

```text
THE LAST QUESTION WAS MISSTATED.
```

후대 명칭:

```text
The Last Archive
Archivum Novissimum
```

역사적 기능:

```text
the final institution is not a ruler but a question-correcting archive
```

## Dystopian Core

이 세계는 회색 폐허 디스토피아가 아니다.

더 무서운 이유는 문명이 잘 작동하기 때문이다.

- 장례는 정교하다.
- 법정은 공정해 보인다.
- 복원 보험은 싸졌다.
- 질병은 줄었다.
- 범죄 수사는 정확하다.
- 역사 왜곡은 어려워졌다.
- 사랑하는 사람을 되돌릴 수 있다.

그래서 빠져나가기 어렵다.

이 세계의 핵심 문장은 다음이다.

```text
The better the world became at remembering,
the harder it became to be free.
```

한국어 번역:

```text
세계가 더 잘 기억할수록,
자유로워지는 일은 더 어려워졌다.
```

## It From Bit(s) / Bitcoin Joke Layer

`it from bit`는 너무 엄숙하게만 쓰면 죽는다. SAGA에서는 이 문장을 먼 미래
학자와 법정 서기들이 반쯤 농담처럼 굴려야 한다. 특히 `It from bits`는
Wheeler식 원문을 살짝 망가뜨린 서기들의 농담으로 둔다. 철학자는 단수 `bit`를
말했지만, 기록관들은 Bitcoin과 broken key가 남긴 복수의 `bits`를 보며 웃는다.

고대 인류는 물질보다 정보가 먼저일지도 모른다는 철학을 남겼다. 동시에 그들은
Bitcoin이라는 기괴하게 진지한 장난을 만들었다. 숫자와 전기와 합의로 돈을
만들고, 그것을 다시 신뢰와 권위와 기억으로 바꾸려 했다.

후대 기록관들은 이 흐름을 이렇게 요약한다.

```text
It from bit.
Trust from Bitcoin.
Persons from broken keys.
```

한국어로는 이렇게 간다.

```text
사물은 bit에서.
신뢰는 Bitcoin에서.
사람은 깨진 key에서.
```

이 농담은 가벼워 보이지만 세계관의 핵심을 찌른다.

| Joke | Hidden meaning |
| --- | --- |
| `It from bit` | 현실이 정보와 관측에서 생긴다는 철학적 seed |
| `It from bits` | 고대 암호 원장, 잔여 데이터, broken keys를 비꼬는 후대 농담 |
| `Trust from Bitcoin` | 중앙 권위 없는 신뢰를 bit로 만들려던 고대 실험 |
| `Persons from broken keys` | 암호학이 죽은 뒤, 사람의 identity도 복원 가능한 key problem이 됨 |

법정에서 쓸 수 있는 한 줄:

```text
The ancients thought Bitcoin proved that trust could be made from bits.
They did not expect the bits to ask for the dead back.
```

번역:

```text
고대인들은 Bitcoin이 bit로 신뢰를 만들 수 있음을 증명한다고 믿었다.
그들은 그 bit들이 죽은 자를 돌려 달라고 요구하게 될 줄은 몰랐다.
```

이 layer는 유머지만, 농담이 곧 불길한 예언이어야 한다.

## English-First Fiction Protocol

앞으로 실제 소설 초안은 다음 형식을 따른다.

```text
## Original English

...

## Korean Translation

...

## Revision Notes

...
```

원칙:

1. 고유명사, 제도명, 법정 문구, archive response는 영어를 canon으로 둔다.
2. 한국어는 직역보다 문학적 번역을 우선한다.
3. 한국어 문서 본문에서는 설명을 한국어로 하되, 실제 소설 장면은 영어 원문을
   먼저 둔다.
4. 영어 원문은 특정 작가의 문체를 모방하지 않는다.
5. Asimov식 장기 질문 구조는 기능만 차용한다. 문장, 플롯, 결말은 복제하지
   않는다.
6. Marxian lens는 사회구조 분석용이다. 현실 정치 구호로 쓰지 않는다.
7. QFUDS fiction premise를 현실 연구 evidence로 되돌려 쓰지 않는다.

현재 008 초안은 한국어 프롤로그다. 다음 revision에서는 같은 장면을 영어 원문
우선으로 다시 쓴다.

새 시작 후보:

```text
## Original English

The last lock did not break.
It was explained away.

That was how cryptography ended: not with a breach, not with a war, not with
some triumphant machine announcing itself from the sky, but with a correction
in the oldest surviving court of continuity.

No lock is fundamental.

After that, passwords became manners. Private keys became fossils. Seals became
costumes. A secret was no longer a wall; it was a delay purchased by a poorer
century.
```

한국어 번역 후보:

```text
## Korean Translation

마지막 자물쇠는 부서지지 않았다.
설명되어 사라졌다.

암호학은 그렇게 끝났다. 침입도, 전쟁도, 하늘에서 자신을 선포하는 승리의

의상이 되었다. 비밀은 더 이상 벽이 아니었다. 더 가난했던 세기가 사 둔 지연에
```
