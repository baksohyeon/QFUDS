# QFUDS SAGA 단편 S1 반려 버튼 Intent Card

## Boundary

```text
fiction/provenance only
research evidence: no
canon status: draft only
```

S1은 별도 anthology가 아니다. SAGA 내부 단편 연작의 첫 후보이며, 1.5부 사엘 이후
검증경제 말단에 놓는 작은 행정 사건이다. 사엘을 재등장시키지 않는다. 사엘의 선례가
양식과 말단 절차로 굳은 뒤, 그 양식 아래에서 새 인물이 한 번 누르는 버튼을 쓴다.

## Story Contract

| 항목 | 값 |
| --- | --- |
| Working title | 반려 버튼 |
| POV | 민재, 접수 심리실 말단 검증관 |
| Scene limit | 버튼 하나, 양식 하나, 가족 한 팀 |
| Immediate want | 민재는 비동의 사망자의 복원 청구를 반려해 절차를 지키려 한다 |
| Counterpressure | 유족은 장례·상속·거주 보조를 풀기 위해 승인 도장이 필요하다 |
| Ticking clock | 반려 입력 전까지 보조금 보류 경고가 갱신된다 |
| Option A | 반려한다. 죽은 사람의 비동의는 남지만 산 가족의 보조가 멈춘다 |
| Option B | 승인한다. 보조는 풀리지만 복원 불가자가 청구 가능 자산으로 바뀐다 |
| Final action | 반려 버튼을 누른다 |
| Final cost | 반려는 취소가 아니라 보존 상태 전환이며, 민재의 서명이 기록된다 |

## Reader Onboarding

| Term | In-scene handle |
| --- | --- |
| 복원 청구 | 유족이 들고 온 접수 번호와 화면 |
| 복원 불가자 | 생전 비동의 또는 동의 불명 상태로 보호되는 사망자 |
| 청구 가능 자산 | 승인 도장이 찍히면 장례·상속·거주 보조와 묶이는 행정 상태 |
| 계좌 보류 경고 | 유족 휴대폰의 즉시 알림 |
| 선례 | 양식 하단에 자동 인용되는 문장 |

설명은 장면 안의 문서, 경고문, 도장, 휴대폰 알림에 붙인다. 복원은 부활이 아니다.
사망자의 기록·데이터 재구성 사본으로만 다룬다.

## Continuity Guardrails

- 하드 연도는 쓰지 않는다.
- Q-Day, Last Archive, 제국 정치, 2부 법원급 세계사는 쓰지 않는다.
- 사엘은 직접 등장하지 않는다.
- 복원을 원본 귀환, 영혼 회수, 엔트로피 역전으로 쓰지 않는다.
- 장면의 승부는 철학 토론이 아니라 화면 상태 변경이다.

## Draft Hook

```text
반려 버튼은 빨갛지 않았다. 너무 많이 눌린 버튼은 색을 잃고, 사람의 죽음보다 더
사무적인 회색으로 변했다.
```

## Ending Target

```text
반려 완료
재심 대기
생계지원 보류 유지
담당 검증관 서명 등록
```

닫힌 사건이어야 하지만 해결이어서는 안 된다. 버튼은 작동한다. 접수는 사라지지 않는다.
