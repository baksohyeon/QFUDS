# QFUDS SAGA 단편 연작 브리프

## 정정된 위치

이 문서는 별도 QFUDS Verse anthology가 아니다. **QFUDS SAGA 내부의 단편 연작
작업선**이다.

목표는 SAGA를 장편 원고 한 덩어리로 완성하는 것이 아니다. 각 단편은 한 사건·한
인물·한 선택으로 닫되, 단편들이 누적되면 SAGA 연속성의 장편 효과를 만든다.

```text
단편 하나 = 독립 사건
단편 여러 개 = SAGA 연속성
```

## Working Principle

- SAGA를 폐기하지 않는다.
- 장편 원고처럼 0부→1부→1.5부→2부를 한 번에 밀지 않는다.
- 각 단편은 SAGA 세계 안의 작은 사건으로 쓴다.
- 단편은 독립적으로 읽히되, 반복되는 제도·도장·양식·질문이 축적되어 SAGA의 큰
  질문으로 이어진다.
- SAGA 주요 인물은 처음부터 쓰지 않는다. 새 인물과 주변 사건으로 시작한다.
- 단편이 쌓여 canon 후보가 되면 chronicler pass로 209·306·310·413에 연결한다.

## Craft Harness Applied

| 항목 | 적용 |
| --- | --- |
| Premise | 복원·검증·상속·기록이 발달한 SAGA 세계에서, 한 사람이 상실을 어떻게 지키거나 빼앗기는지 본다 |
| Character | 매 편 새 focal character 우선. Want/Need/Fear는 intent card에서 고정 |
| Conflict | 사적 애도와 제도적 청구가 충돌한다 |
| Stakes | 신원, 장례, 상속, 가족관계, 접근권, 기록 삭제, 복원 거부 |
| POV | 단편마다 한 focalizer 또는 하나의 인월드 문서 형식 |
| Technical grounding | 기술어 보존. 설명은 화면·양식·도장·경고문·법정 기록에 붙인다 |
| Korean mode | 한국어 primary. 외국어식 로그라인을 번역하지 않는다 |
| Boundary | fiction/provenance only. 연구 증거나 보안·법률 조언 아님 |

## Collection Rule

```text
한 사람.
한 양식.
한 선택.
한 번 되돌릴 수 없는 비용.
```

| S6 | 이름 셋 중 하나 | 2부 하류 구역 주변 | 하류 구역 현장 기록 | 이름을 흩뜨려 살아온 사람이 한 이름으로 증언해야 한다 | Noor 직접 사용 금지. 새 인물 |
| S7 | 삭제 신청서 | 복원 행정 초기 사건 | 행정 양식 + 직접 장면 | 가족은 죽은 사람을 복원하려 하고, 본인은 생전 삭제 신청서를 남겼다 | 철학 논쟁 금지. 신청서 한 줄 |
| S8 | 마지막 원본 | 먼 미래 수장고 | 박물관 수장고 기록 | 모두가 사본으로 사는 시대, 한 낡은 물건이 원본이라는 이유로 거래된다 | 세계사 요약 금지. 물건 하나 |

## First Pick Recommendation

첫 단편은 S1 `반려 버튼`이 가장 작다.

이유:

- SAGA 핵심인 검증·복원·상실권을 바로 보여 준다.
- 기존 주역 없이도 작동한다.
- 단편 하나만 읽어도 닫힌다.
- 나중에 사엘 origin, 2부 Mara, 연속성 법원으로 연결할 수 있다.

## Canon Handling

단편 초안은 바로 canon이 아니다.

```text
draft -> review -> chronicler pass -> canon candidate -> 209/306/310/413 반영 여부 결정
```

단편이 기존 SAGA 사건과 연결될 때만 해당 문서에 역배선한다. 연결 전에는
`20_drafts/단편집/`의 독립 원고로 보존한다.
