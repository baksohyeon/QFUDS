# QFUDS Verse World

이 폴더는 `qfuds-verse`의 공통 세계관 규칙을 둔다.

## World Premise

먼 미래 문명은 정보의 소실, 복원, 암호학 붕괴, post-AGI 통치, 기억과
망각권을 둘러싸고 제도와 신화를 만든다.

QFUDS는 여기서 연구 결론이 아니라 fiction premise다. 어떤 story layer에서
QFUDS가 맞다고 가정할 수 있지만, 그 가정은 소설 장치다.

## Technical Rule

기술어는 보존한다.

| Term | Rule |
| --- | --- |
| `hash`, `KDF`, `key`, `salt`, `collision` | 암호학 개념으로 유지 |
| `entropy`, `Hawking radiation`, `Page curve`, `island` | 과학/철학 장면에서 정확히 설명 |
| `AGI`, `QFUDS` | 설정 장치와 연구 경계를 분리 |
| Bitcoin / Genesis Chain | artifact와 신화로 사용 가능 |
