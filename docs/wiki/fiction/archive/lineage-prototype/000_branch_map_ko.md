# 분기 지도

## 먼저 읽을 것

이 문서는 QFUDS Fiction Saga의 인터렉티브 분기 지도다.

연구 결과가 아니다. 물리 증거가 아니다. 독자가 어떤 SF 경로를 따라 읽을지
고르는 worldbuilding 장치다.

## Workflow Boundary

This branching map does not introduce a new external web, PDF, table, product,
cache, digitization, or availability claim. It organizes fiction routes derived
from existing QFUDS lineage documents only.

External-source handling remains governed by
[Research Asset and Product Workflow](../../../../../.agent/workflows/research-asset-product-workflow.md).
Inherited workflow states include:

```text
asset_cached
asset_extracted_not_digitized
source_text_parse
manual_structured_extract
inspected_no_numerical_product
```

## 읽는 법

각 에피소드 끝에는 선택지가 있다.

선택지는 실제 연구 결정이 아니다. 이야기의 경로다. 다만 각 선택지는 QFUDS가
실제로 부딪힌 과학적 갈림길을 반영한다.

```text
A route: black-hole / remnant / information return
B route: galaxy-scale foam / 10 Mpc / known-model sink
C route: Gamma(a) / IV/IDE formalism / Q^mu grammar
D route: audit harness / postmortem / failed-claim memory
E route: academy orders / forbidden rooms / initiation rites
```

## 시즌 1 메인 분기

### A. 블랙홀 우편함

질문:

```text
블랙홀이 삼킨 정보가 사라지지 않는다면,
그 정보는 어떤 봉투로 돌아오는가?
```

주요 인물:

- 노아 백
- 한서윤
- 감사관

위험:

```text
remnant를 너무 쉽게 쓰면 compact-object constraints에 걸린다.
정보 보존을 곧바로 dark-energy source로 쓰면 physics_blocked다.
```

현재 에피소드:

- [제2화: 화이트홀 우편함](002_white_hole_mailbox_ko.md)

## 독자 선택 메모

재미로 읽는다면 A route부터 가면 된다. 블랙홀, remnant, white-hole-like
mailbox가 가장 SF적이다.

과학 문법을 따라가고 싶다면 C route가 좋다. `Gamma(a)`가 왜 `Q^mu`가 아닌지,
왜 formalism이 필요한지 드러난다.

세계관의 큰 배경을 보고 싶다면 B route가 좋다. 은하 규모 foam과 cosmic web이
장면으로 살아난다.

이 repo의 진짜 핵심을 보고 싶다면 D route가 좋다. 정답보다 실패를 보존하는
하네스가 왜 남았는지 보여 준다.

더 입체적인 saga 세계관을 원한다면 E route가 좋다. 라우어 관측소가 단순한
연구기관이 아니라 학파, 금지구역, 입문 의식을 가진 비밀 아카데미로 확장된다.
