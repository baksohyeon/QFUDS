# Blocked Admission Rule Checkpoint

## Current Gate

No Physical-QFUDS [Level 2B](../glossary/repository_levels.md) work is allowed until all five items are supplied by
one mechanism.

This checkpoint restates the admission rule for handoff clarity. It is not a
roadmap status table. Current status remains governed by
[docs/05_next_steps/000_roadmap.md](../../05_next_steps/000_roadmap.md).

## Blocked Items

```text
X =
Q^nu =
why phase B has w ~= -1 =
delta Q route =
known-model distinction =
```

All five items must be supplied together by one candidate mechanism. A partial
answer, analogy, timing coincidence, or renamed coupling function does not admit
a physical branch.

## Non-Admission Rules

Do not open a physical branch from:

- Landauer motivation alone;
- generic quantum foam language;
- white-hole or remnant language without equations;
- retained `Gamma(a)` timing alone;
- a conservation-compatible `Q^nu` ansatz that lacks a physical source,
  phase-B rationale, and `delta Q` route.

Any future candidate must introduce an equation, observable, changed viability
decision, or reproducible experiment before it can move out of the audit lane.
