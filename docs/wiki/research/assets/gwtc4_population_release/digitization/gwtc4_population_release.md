# GWTC-4.0 Population Release — Data Release Document

MarkItDown-based markdown document for the LIGO-Virgo-KAGRA data release
"GWTC-4.0: Population Properties of Merging Compact Binaries" (Zenodo record
16911563). This asset is a **data release**, not a single paper PDF: the source
cache holds the release notes, figure PDFs, figure-generation scripts, event
lists, and a tutorial notebook, but not a paper manuscript PDF.

The release notes and tutorial were converted with
[MarkItDown](https://github.com/microsoft/markitdown); the figures are the
rendered PNG mirrors under `figures/extracted/`. Quality: `source_text_parse`.

Converted sources: [release notes](zenodo_README.md) ·
[popsummary tutorial](popsummary_tutorial.md).
