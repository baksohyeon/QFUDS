Supplementary material to DESI's publication DESI DR2 Results I: Baryon Acoustic Oscillations from the Lyman Alpha Forest. The .tgz file contains all of the data needed to reproduce the figures in the paper, as required by the DESI Data Management Plan. There is also a README with additional details on file format.
