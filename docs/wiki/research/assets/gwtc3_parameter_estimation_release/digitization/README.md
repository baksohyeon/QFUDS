# GWTC-3 Parameter-Estimation Release Digitization Assets

This folder stores derived inspection products for the GWTC-3 parameter-estimation
data release (Zenodo 8177023). This asset has no paper, text, or HTML source to
transcribe — only the Zenodo record JSON — so the "document" here is a Markdown
