# Amendola 2024 Digitization Assets

This folder stores derived inspection products for the Amendola 2024 asset.

The full-text Markdown was regenerated via [PageIndex](https://github.com/VectifyAI/PageIndex) MCP (`source_text_parse`),
replacing the earlier low-fidelity [MarkItDown](https://github.com/microsoft/markitdown) pass. It is a faithful page-by-page
text and LaTeX-equation transcription (including Table 1 and Eqs. 1–23, B1–B4)
with figure image links wired to the repo PNG mirrors under
`../figures/extracted/plots/`. It is machine-extracted and not line-for-line
verified, so it must not be treated as numerical digitization or audited equation
provenance.

For GW constraint work, prefer source/code reproduction where available. If the
audit needs paper-level values, manually extract only the needed equations,
tables, captions, or figure values from the source PDF, TeX, figures, or
rendered figure PNGs, then store the curated output here.
