# Li 2025 Digitization Assets

This folder stores derived inspection products for the Li and Zhang 2025 asset.

The full-text Markdown was regenerated via PageIndex MCP (`source_text_parse`),
replacing the earlier low-fidelity MarkItDown pass. It is a faithful page-by-page
text and LaTeX-equation transcription of the main body and appendices (pages
1–14, including Tables 1–3 and Eqs. 1–10) with figure image links wired to the
repo PNG mirrors under `../figures/extracted/`. The bibliography (pages 14–20,
118 refs) is only partially captured. It is machine-extracted and not
line-for-line verified, so it must not be treated as numerical digitization or
audited equation provenance.

For timing-overlap work, manually extract only the needed items from the source
PDF, TeX, figures, or rendered figure PNGs, then store the curated output here.
