# Kasen 2017 Kilonova R-Process GW170817

## Bibliographic Metadata

- Paper: "Origin of the heavy elements in binary neutron-star mergers from a
  gravitational wave event"
- Authors: Daniel Kasen, Brian Metzger, Jennifer Barnes, Eliot Quataert,
  Enrico Ramirez-Ruiz
- arXiv: [1710.05463](https://arxiv.org/abs/1710.05463)
- Journal reference: Nature 551, 80-84 (2017)

## Key Equations

- Models kilonova emission from r-process ejecta in GW170817.
- Uses optical/infrared observations to infer ejecta components, masses,
  velocities, and compositions.

## Coupling Definitions

- No dark-sector coupling is defined.
- Candidate source variables: kilonova ejecta mass, r-process production per
  merger, merger-rate-scaled r-process production history.

## Datasets Used

- GW170817/AT2017gfo optical and infrared observations.
- Kilonova radiation-transport model comparisons.

## Redshift Coverage

- Single nearby event, not a redshift-history product.
- A cosmic production history would require convolution with BNS/NSBH merger
  rates.

  pass.
