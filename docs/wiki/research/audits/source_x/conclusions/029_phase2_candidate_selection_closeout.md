# 2026-06-10 Phase 2 Candidate Selection Closeout

## Purpose

This document closes Phase 2 of the Source-X audit sequence before entering
Phase 3.

Phase 2 evaluated Source-X candidate lanes. It used the Source-X audit, the
black-hole-coupled source audit, the repaired black-hole literature cache, and
the expanded structure-era activation coverage to decide which lane should be
carried into a Phase 3 `Q^nu` feasibility / derivation attempt.

This closeout is not an experiment result, roadmap update, or physical-branch
admission.

## Phase 2 Decision

Decision: retain the black-hole-coupled lane as the Phase 3 audit candidate.

The black-hole-coupled lane is retained because it is closest to the original
QFUDS strong-gravity intuition and has the strongest known-model reduction
pressure. It is the lane most likely to clarify whether the strong-gravity
intuition collapses into cosmologically coupled black-hole dark energy,
black-hole-coupled dark energy, black-hole entropy/topological dark energy,
remnant dark matter, or a selected interacting-vacuum / IDE coupling history.

This retained Phase 3 candidate is not an admitted physical source `X`.

The black-hole-coupled lane is retained for audit pressure only. It is not
retained because black holes explain `Gamma(a)`, because `Gamma(a)` has been
proven, or because the literature cache supplies a complete QFUDS model.

## Non-Primary Lanes

Compact-object and structure-era lanes remain comparison or coverage-expansion
lanes, not Phase 3 primary candidates.

The compact-object transient lane is useful for checking source histories such
as core-collapse supernovae, remnant formation, compact-object mergers,
gravitational-wave backgrounds, and r-process/kilonova production. It does not
currently supply a primary Phase 3 `Q^nu` candidate.

The structure-era activation lane is important because it exposed a
search-keyword gap in black-hole-centered coverage. It keeps backreaction,
vacuum activation, emergent/transition dark energy, running vacuum, and
nonparametric/thresholded IDE on the comparison map. It does not replace the
Phase 3 black-hole-coupled audit candidate because the requested Phase 3 route
is to stress-test the original strong-gravity lane first.

## Phase 3 Entry Rule

Phase 3 may only proceed as a `Q^nu` feasibility / derivation attempt plan, with
failure criteria stated first.

At minimum, the Phase 3 plan must fail the candidate if:

- it only names `X(a)` or a source density without a covariant transfer vector;
- it only supplies a scalar rate `Q(a)` without frame choice, four-velocity, and
  stress-energy split;
- it lacks a normalization that is fixed before fitting;
- it cannot explain why the receiving phase has smooth `w ~= -1`;
- it cannot state a `delta Q` perturbation route;
- it reduces to cosmologically coupled black-hole dark energy, black-hole
  entropy/topological dark energy, remnant dark matter, or generic IV/IDE
  without a distinct QFUDS equation or observable.

Timing overlap with retained `Gamma(a)` is not sufficient.
