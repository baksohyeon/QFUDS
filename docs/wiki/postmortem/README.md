# Wiki Postmortems

This folder stores incident retrospectives and process-failure records.

Use it when the repository needs to preserve how an agent or workflow failed,
what was repaired, and what guardrail should prevent recurrence. Postmortems do
not change roadmap status, promote evidence, or open a new research level.
