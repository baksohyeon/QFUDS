# Wiki Index

This wiki stores research support material that should not live inside the
roadmap, theory docs, experiment docs, or result docs.

The roadmap remains the status authority:
[docs/05_next_steps/000_roadmap.md](../05_next_steps/000_roadmap.md).

Agent operational workflows are not maintained here. Future agents must use
[.agent/README.md](../../.agent/README.md) and
[.agent/workflows/README.md](../../.agent/workflows/README.md) for executable
workflow rules, including the
[Documentation Folder Routing Workflow](../../.agent/workflows/documentation-folder-routing-workflow.md).

## Folder Roles

- [glossary/](glossary/README.md): stable vocabulary and repository terminology.
- [governance/](governance/README.md): admission rules, branch gates, consistency checks, and

- [lineage/](lineage/README.md): idea genealogy and branch dependency provenance.
- [postmortem/](postmortem/README.md): incident retrospectives and process failures.
- [research/](research/README.md): external literature records, investigations, cached assets, and
