# Plan 18-01: Add Korean-primary first arc read order

## Objective

Make Korean the active first-arc reading path for `The Broken Crown` while
preserving the existing English v2 drafts as counterparts.

## Scope

In scope:

- Update fiction harness and SAGA protocol docs from English-first to
  Korean-primary.
- Add a first-arc polish/read-order control document.
- Add Korean adaptations for revised Episodes 1-6.
- Update SAGA read order and wiki content index.
- Validate, commit, and fast-forward merge into
  `codex/qfuds-saga-fiction-archive`.

Out of scope:

- Rewriting existing English v2 drafts.
- Arc two planning.
- QFUDS research claims, physical-source claims, Bitcoin prediction, legal
  advice, cryptographic advice, black-hole physics claims, or Level 2B
  language.

## Acceptance Criteria

- Active prose workflow says Korean primary draft first, English Anglophone
  adaptation second.
- Korean read order lists Episodes 1-6 before English counterparts.
- All six Korean adaptation files have valid frontmatter and fiction/provenance
  boundaries.
- Field marks remain aligned:
  `RECOVERABLE / NOT CLAIMABLE`, `ACCESS != AUTHORITY`,
  `NO CONSENT BY ANALOGY`, `PLURALITY IS NOT CONSENT`,
  `PHYSICS IS NOT JURISDICTION`, and `who may author loss`.
- Existing English v2 files remain unchanged.
- Validation and staged workflow guard pass before commit.
