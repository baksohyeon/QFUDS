# Plan 14-01: Add first arc full revision pass

## Objective

Create a revision control document for the completed first arc rough draft set
so the next prose rewrite loops can tighten the arc without losing the working
story engine.

## Scope

In scope:

- Add a new `30_drafts/` revision pass document.
- Preserve all rough draft files as-is.
- Update SAGA README, wiki index, and GSD planning state.

Out of scope:

- Rewriting the episode prose files in this loop.
- QFUDS research claims.
- External literature or web research.
- Science validation language.
- Arc two planning.

## Acceptance Criteria

- The pass gives arc-level and episode-level revision decisions.
- The pass identifies cuts, additions, continuity fixes, and style rules.
- The next rewrite loop is explicit.
- Repository validation and staged workflow guard pass before commit.
