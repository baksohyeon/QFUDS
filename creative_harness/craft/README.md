특정 작품의 상태나 canon은 기록하지 않는다.

- [SF 창작 문예 하네스](creative_writing_craft_harness.md): 전제, 인물, 장면,
  workshop, revision의 core SSOT
- [SF 하위 장르 선택형 렌즈](sf_genre_lenses.md): hard, social, literary,
  horror, technothriller 작품별 질문
- [대학·SF 창작교육 참고 매트릭스](university_creative_writing_reference_matrix.md):
  출처, 허용·금지 주장
- [독자 온보딩](reader_onboarding_harness.md): 낯선 개념의 정보 설계
- [한국어 문장 자연스러움](korean_fiction_prose_naturalness_harness.md):
  한국어를 초고 언어로 고른 작품의 final prose module
- [SF 장면 핍진성](prose_verisimilitude_audit_checklist.md): 기술·제도·일상 인과 감사

`craft_and_political_theory_research.md`는 qfuds-verse에서 파생된 과거 작품별
reference다. 범용 하네스의 필수 읽기가 아니며, 새 작품은 해당 연구를 자동 상속하지
않는다.
