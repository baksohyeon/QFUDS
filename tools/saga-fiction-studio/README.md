# saga-fiction-studio (legacy QFUDS profile)

This plugin is a historical SAGA-specific adapter. It is not the generic fiction
studio and must not be used as the authority for new works.

Active generic authority:

- `.agents/skills/fiction-production/SKILL.md`
- `.agent/workflows/fiction-ip-management-workflow.md`
- `.agent/workflows/agentic-fiction-production-workflow.md`
- `creative_harness/craft/creative_writing_craft_harness.md`

`tools/story-skills` is the optional deterministic long-form continuity engine.

The skills in this directory preserve old role names and QFUDS profile behavior for
historical compatibility. Four skills were reconstructed from UI descriptions, so
they are not a fidelity-authoritative copy of an external plugin. Do not install or
upload this directory for a new project. If a QFUDS-derived work explicitly revives
the profile, inspect and migrate each skill against the active generic workflow
first.
