# QFUDS Verse 세계 확장 웨이브 3 (지리 상세·궤도 역학·사건 연쇄 인과)

## 이 문서의 자리

[117 웨이브 1](117_world_expansion_wave1_names_places_events_ko.md)이 지명·성계를,
[118 웨이브 2](118_world_expansion_wave2_factions_relationships_ko.md)가 세력 관계망을
깔았다. 웨이브 3은 (1) 그 지명에 **내부 구획·궤도 역학·이동 경로**를 넣어 지형이
정치를 낳게 하고(듄의 사막처럼), (2) 웨이브 2 관계망이 만들어내는 **구체 사건의 연쇄
인과**를 짠다. 사건은 기존 캐논 노드(사엘 선례·마라 거부·관측 전쟁·망각권 혁명)로
흘러드는 연결 조직이며, 캐논을 덮지 않는다.

**레벨: universe 공유 세계 후보 register.** canon 아님. 지명 상세는 111 컴펜디움으로,
사건은 115/116/001로 웨이브별 승격한다(117 §7 6체크 기준). 승격 결정: 웨이브 1·2와
동일하게 **원고 수요 때 개별 승격**(사용자 2026-07-01).

## 1. 지리 상세: 지형이 정치를 낳는다

핵심 원리: **검증 밀도가 지형을 계급으로 가른다.** 고밀도 인증 핵심 권역과 인증 사막
변경의 분화(115 C patchwork)가 물리 지리로 굳는다.

### 1.1 지구권 도시 구획

| 도시(117) | 내부 구획 | 정치 함의 |
| --- | --- | --- |
| 베리디온(Veridion) | 초록 광장(진본 인증 중심) / 회색 외곽(미검증 빈민) / 도장 회랑(길드 지대) | 초록↔회색 경계가 곧 시민권 경계. 검증관 길드가 회랑을 지대로 수취 |
| 시그나(Signa) | 물류 결절 / 도장 시장 / 콜드체인 창고 | 도장값이 물값보다 비싼 도시. 재고 소진기(115 A.5) 권력 거점의 원형 |
| 아르젠타 성채(Argenta Keep) | 원장 심부 / 실물 금고 / 침묵 구역 | Domus Ferrata 본성. 지나치게 조용함(정정 불가 원장의 공포, 102) |
| 포르타 클라비움(Porta Clavium) | 관문 심사대 / 회색 대기소 / 통행세 사무 | Domus Repagula 관문. 인증 병목이 곧 통행세. 회색 대기소가 디아스포라 저수지 |
| 살리나(Salina) | 소금 기록고 / witness chain 홀 / 평신도 계보 구역 | Ordo Salis 수도 도시. 공개 기록의 성소. 베이르 가문 계보 연결은 확정(118 §1.4, 2026-07-06). 도시 자체는 candidate 유지 |
| 리미넬(Liminel) | 복원 대기실 / 사후동의 공증소 / 기억 식당 / 저해상 예배당 | 연속성 법원 옆. 경제가 기다림으로 돈다(102). 대기가 계급을 만든다 |

### 1.2 궤도 역학

| 궤도 도시(117) | 궤도 위치 | 병목·이동 |
| --- | --- | --- |
| 하이 레저(High Ledger) | 지구 정지궤도 | exit 계급의 첫 정거장. 지구→궤도 상승 비용이 계급 필터 |
| 키네리아(Cineria) | 달 재의 평원 | Domus Cinerum 죽은 지갑 아카이브·재 광산. 저중력 채굴 |
| 테라 노비시마(Terra Novissima) | 화성 | 부자 먼저 이주(111 §1). 지구→화성 전이창이 이주 계급을 배급 |
| 오르비스 무티(Orbis Muti) | 라그랑주 | "침묵의 궤도". 인증 없이 도는 회색 정거장. 관할 밖 회색 경제 |

- **전이창 정치:** 궤도 도시 간 이동은 연속적이 아니라 전이창(주기적 발사 기회)에 묶인다.
  전이창 배급권을 쥔 자가 exit를 배급한다(exit>voice의 궤도판, 115 C). 하이 레저가
  첫 관문.
- **섬화(islanding)와 궤도:** Q-Day 직후 국지 섬화(115 A)의 우주판. 인증이 끊긴 궤도
  도시는 물리적으로 고립된다. 오르비스 무티가 그 회색 고립을 상시화한 곳.

### 1.3 변경·유목 이동 경로·코어

```text
[핵심 권역]  베리디온─High Ledger─Terra Novissima   고밀도 인증, 초록 질서
     │  (검증 밀도 구배)
[변경]      그레이마치─Ager Sine Sigillo             인증 사막, 점유=소유
     │
[유목 경로]  재의 초원─금기된 우물─혜성 궤도          잔상 밀도장을 따라 이동
     │
[코어]       Ultima Statio (블랙홀 궤도)              Last Archive 본거지, 열이 죄
```

- **유목 목초지 = 잔상 밀도장:** 잔상 유목민은 행성이 아니라 정보 밀도장을 따라 이동한다
  (102). 전쟁터·폐관측소·혜성 궤도가 목초지. 재의 서약장 아쉬켄(118)이 경로를 성력으로.
- **코어 접근 금기:** Ultima Statio는 열이 죄인 저온 계산권역(102). 접근 자체가 의례.
  우물 관측단(118)이 그 잔상 상관 복원의 미결정을 품는다(113·114 물리 경계 유지).

### 1.4 지형 = 정치 요약표

| 지형 | 통제 자원 | 낳는 권력 |
| --- | --- | --- |
| 초록 광장 핵심 권역 | 인증 밀도 | 검증관 길드 지대, 시민권 경계 |
| 관문·전이창 | 이동 병목 | Domus Repagula 통행세, exit 배급 |
| 인증 사막 변경 | 점유=소유 | 지하 인증 카르텔, 자경, 이념 결사 |
| 잔상 밀도장 | 죽은 기억 | 유목 oath 부족, 잔상 암시장 |
| 블랙홀 코어 | 복원 계산 | Last Archive 신격화 |

## 2. 사건 연쇄 인과: 관계망이 낳는 구체 사건

웨이브 2 관계 매트릭스(118 §2.2)가 만들어내는 사건 사슬. 전부 후보이며, 끝에서 기존
캐논 노드로 흘러든다. 화살표(→)는 한 압력이 다음을 낳는다.

### 2.1 잔상 몰수 사슬 (부채 → 예속 → 계급 고착)

```text
[118] Vadis(베라 다체)가 Fossoria(세라핀 반)에 잔상 채굴 자금 대출
  → 채굴 잔상을 담보로 (잔상 소유권 담보화)
  → 채굴 부진·디폴트
  → 잔상 소유권이 Vadis로 이전 (신흥 발굴 가문이 담보 가문에 예속)
  → "잔상 몰수" 행정 사건(후보): 발굴자 본인은 조립되지 않고 부자 복원 재료로만 쓰임
  → [캐논 흐름] 데이터 식민주의(115 C)·해상도 계급 고착. 신체 호러 아닌 등기·참조 형식
```

### 2.2 도장 스캔들 사슬 (사랑 → 은폐 → 감사 붕괴 위협)

```text
[118] 파울 렌(Ordo Circuli)이 사랑하는 이를 되살리려 감사 도장을 건너뜀
  → circular_if_fitted(결과 보고 원인 맞춤) 은폐
  → 이스마 콜(Camera Ignotorum)이 "오늘의 unknown"에서 균열 발견
  → "도장 스캔들"(후보): 감사 도장이 사랑으로 오염된 사례가 공론화
  → [캐논 흐름] 인간 확인 루프의 거수기화 위협(112). 리오라 mark의 미래 권력화 전조
```

### 2.3 명부 소각 사슬 (이탈 → 충돌 → 혁명 불씨)

```text
[118] 잿빛 손(회색 이탈)이 절차를 버리고 자경으로
  → 셀레(붉은 명부단, 적)와 카이(검은 침묵단, 흑)의 충돌 격화
  → 이오네가 분열 선택을 강요당함(209 결정 필요)
  → "명부 소각"(후보): 지워진 자 명부를 두고 적·흑이 물리 충돌
  → [캐논 흐름] 6기 망각권 혁명의 불씨(102·115 D). 잊힐 권리조차 계급이라는 비관
```

### 2.4 회색 범람 사슬 (통행세 → 우회 → 행정 마비)

```text
[118] Repagula(할브렉트)가 관문 통행세를 과다 인상
  → 먼지길 대상단이 미인증 회색 인구를 우회로로 실어나름
  → 포르타 클라비움 관문의 회색 대기소 범람
  → "회색 범람" 행정 사건(후보): 진본 신원 증명 못 하는 인구 일괄 "회색" 처리
  → [캐논 흐름] 미인증 디아스포라·Bloom 필터 비극(115 B). 무국적이 기본값
```

### 2.5 묘비 상속 판례 사슬 (분쟁 → 폭로 → 선례)

```text
[118] 6가문 상속 분쟁 (Aurata-Vadis 혼담 결렬 등)
  → Cinerum(톰 로익)이 묘비 자산 세탁 내역으로 편을 바꿈(약점을 쥔 회색 위치)
  → 연속성 법원(Curia Continuum) 심리
  → "묘비 상속 판례"(후보): 움직여 죽은 지갑(묘비)의 상속권 판정
  → [캐논 흐름] 「복원 가능=청구 가능」 사다리(115 §3.4)의 한 칸. 소송이 곧 GDP
```
