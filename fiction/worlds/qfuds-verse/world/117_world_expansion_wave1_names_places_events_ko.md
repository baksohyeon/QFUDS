# QFUDS Verse 세계 확장 웨이브 1 (고유명사·지명·세력·인물·사건·어휘)

## 이 문서의 자리

듄급 세계 밀도를 목표로 한 **대량 고유명사 생성 1차 웨이브**다. 세계를 "이름표"가
아니라 살아 있는 밀도로 키우기 위해, Q-Day 이후 문명의 세력·가문·인물·지명·성계·
사건·어휘를 대량으로 짓는다.

**레벨: universe 공유 세계(`qfuds-verse/10_world/`).** 이 대사전은 한 시리즈의 플롯
자료가 아니라 universe 전체가 공유하는 세계 자료라, 시리즈(qfuds-saga) 안이 아니라
universe 레벨에 둔다. 세부 세계 캐논은 현재 qfuds-saga series bible이 보유하며(10_world
README), 이 register는 그 위에 밀도를 얹어 안정되면 series bible SSOT로 승격한다.

**거버넌스: 이건 아직 canon이 아니다.** universe 공유 세계 **후보 register**이며,
웨이브별로 검토 후 각 권위 bible로 승격한다(명칭=109, 인물=209, 세력·생태=102, 여파
사건=115·116, 전부 `../20_series/qfuds-saga/00_bible/`). momentum 캐논화를 하지 않는다.
채택·승격은 사용자 승인 게이트 뒤에만.

```text
fiction/provenance only

캐논 상태: universe 공유 세계 candidate register. 109·209·102·115·116 승격 대기
```

## 생성 규율 (충돌·일관성 가드)

기존 캐논과 대조해 아래를 지켰다.

1. **기존 고유명사와 충돌 없음.** 109(세력명)·209(인물명)와 대조. 기존 이름
   (Domus Clavium·Bureau Laurien·Sael·Mara Veyr·Elias Veyr·Liora Sen·Pell Anson·
   Noor Aram·Ione·Tamas·Sera Venn·Adrian Karvath·Vera 등)을 재배정하지 않는다.
   신규는 전부 빈 좌표·하위 분파·신규 인물로만 넣는다.
2. **Latin 명명 관습 상속(109).** 제도는 Domus/Ordo/Curia/Camera/Custodes/Schola/
   Cohors/Statio/Claustrum 형식어로 짓는다. 이념 계열은 므네모시네(보존)·레테(망각).
3. **복원=사본 절대(113).** 어떤 복원 기술·시장 이름도 부활이 아니라 손실적 사본.
4. **필드마크 사슬 불변(115 §4).** 6개 mark 순서에 손대지 않는다. 신규 표어·구호는
   사슬 밖 문화 어휘로만 넣는다.
5. **실존 집단 비대응(109·102).** 특정 실존 민족·종교·정당에 직접 대응하지 않는다.
   권력 축은 정보·복원·소유이며 누구에게나 같게 적용한다.
6. **스타일 가드.** em dash 0, "박-" 슬랭 금지, mark 표기 유지, 강의조 금지.

## 1. 명명법 체계 (Naming System): 백본

이름이 일관되려면 문화별 명명 문법이 먼저 있어야 한다. 아래가 이 세계의 명명 뼈대다.

| 문화권 | 명명 문법 | 예시 형태 |
| --- | --- | --- |
| 제도·기관 (Aletheia 후신) | Latin 형식어 + 기능 속격. Domus/Ordo/Curia/Camera/Custodes/Statio/Claustrum | Domus Ferrata, Camera Ignotorum |
| 장부 가문 (Domus Clavium 계) | Domus + 물성/자물쇠/담보 어근. 성씨는 key 지분(cap table)에서 파생 | Domus Vadis, 가문명 Kelver, Orsin |
| 검증도시 시민 (초록 도장 문화) | veritas/signum 어근 도시명. 일상 의례어 "초록/회색" | Veridion, Signa |
| 라우어 감사 계열 | 미지·순환·도장 어휘. Camera/Ordo + 방법론어 | Camera Ignotorum, Ordo Circuli |
| 망각권·레테 계열 | 이름을 지우는 문법. 세 이름 체계(부르는/법적/복원되어도 안 말할 이름) | Domus Sine Nomine, Claustrum Lethes |
| 잔상 유목민 | 서약명(공유 key) + 경로·혜성·잔상밀도 지명. 혈통 대신 oath | 재의 띠 부족, 서약명 Ashkeeper |
| 인증 사막 변경 크레올 | Latin 파편 + 토착 혼종. 회색(미검증) 정체성 | Greymarch, Ager Sine Sigillo |
| 계급어(register) | 고해상어(高, 격식·완전 기록) vs 저해상어(低, 축약·잔상) | 아래 §6 어휘집 |

**성씨 파생 원리(장부 가문):** 성씨는 조상이 Q-Day 때 쥔 지갑·복원 키의 별칭에서
왔다. 큰 가문일수록 오래된 key 별칭을 성으로 쓴다. 가난한 자는 성이 없고 회색
등록번호만 있다("성이 없다"가 곧 계급 표지).

## 2. 세력·가문·분파 (하위 조직 대량)

기존 8세력 아래 하위 가문·분파·수도회·초소를 단다. 전부 후보(102·109 승격 대기).

### 2.1 장부 가문(Domus Clavium) 하위 가문

| 가문명(Latin) | 한국어 | 무엇을 쥐나 | 성격 |
| --- | --- | --- | --- |
| Domus Ferrata | 강철문 가문 | 데이터 성채·물리 금고 인프라 | 가장 오래된 패권 가문, 실물+데이터 |
| Domus Vadis | 담보 가문 | 사후 보험·복원 담보 대출 | 상실을 채권으로 만드는 금융 분파 |
| Domus Aurata | 금장 가문 | 고해상 복원 슬롯 상속권 | 사후 계급의 정점, 가장 화려 |
| Domus Cinerum | 재의 가문 | 주인 없는 죽은 지갑(대탈취 유산) | 묘비 자산을 세탁해 상속으로 |
| Domus Fossoria | 발굴 가문 | preimage 복원 하청·잔상 채굴 | 손 더러운 신흥, 구가문에 멸시받음 |
| Domus Repagula | 빗장 가문 | 국경·궤도 관문 인증 통행세 | 이동을 팔아 부를 쌓는 관문 영주 |

대표 성씨(가문 예속): Kelver, Orsin, Dace, Halbrecht, Vane, Roik. (전부 신규 후보.)

### 2.2 라우어 관측소(Bureau Laurien) 감사 조직

## 6. in-world 어휘집 (Lexicon)

세계의 설득력은 사람들이 쓰는 단어에서 나온다. 계급어·종교어·군사어·저주·인사.
전부 후보(102·111 승격 대기). 기존 캐논 어휘(초록 네모/회색/묘비/해상도 계급/필드마크)
는 유지하고 그 위에 얹는다.

### 6.1 일상·계급어

| 말 | 뜻 |
| --- | --- |
| 초록(green) / 회색(grey) | 진본 / 미검증. 사람에게 쓰면 시민권 유무 |
| 고해상어 / 저해상어 | 격식·완전 기록 계층의 말 / 축약·잔상 계층의 말 |
| 성이 없다(no surname) | 가문 key가 없다 = 회색 등록번호만 있는 빈자 |
| 묘비(tombstone) | 움직여서 죽은 지갑·계정. 비유로 "붙잡혀 막힌 것" |
| 솔기(seam) | 복원 사본에서 어긋나는 지점("솔기가 보인다") |

### 6.2 종교·의례어

| 말 | 뜻 |
| --- | --- |
| 네 도장이 함께하길 | 인사·축복. `defined`/`assumed`/`unknown`/`circular_if_fitted` |
| 오늘의 unknown | 식전 낭독 대신 오늘 모르는 것을 읽는 라우어 의례 |
| checksum이 맞았다 | 라스트 아카이브 계열의 "아멘" 등가, 기억 일치 확인 |
| 정의되지 않은 구원은 폭력 | *캐논 교리문답.* 감사주의 핵심 계율 |
| 세 이름 | 부르는 이름 / 법적 이름 / 복원되어도 안 말할 이름(망각권) |

### 6.3 군사·행정·저주

| 말 | 뜻 |
| --- | --- |
| 진본 인질(hostage-authentic) | 진짜임을 인질로 잡고 자릿세를 걷는 지하 인증 카르텔 전술 |

### 승격 판단 기준 (6체크, 모든 웨이브 공통)

각 후보(이름·인물·세력·사건)를 아래 6개로 걸러 승격을 정한다.

1. **서사 수요(pull):** 지금 어떤 원고·아크가 이걸 실제로 쓰나? 아무도 안 쓰면 보류
   (캐논은 수요가 당긴다).
2. **충돌 위험:** 109/209/102/113·필드마크와 부딪히나? 부딪히면 고친 뒤 승격.
3. **설계 안정성:** 역할·관계가 굳었나? 209 '결정 필요' 항목이면 보류.
4. **의존 파급:** 바뀌면 몇 개가 흔들리나? 세력·명칭은 크므로 확신 있을 때만.
5. **레벨 정합:** universe 공유(102/109)냐 시리즈 전용(209)이냐, 맞는 선반으로만.
6. **게이트 충족:** bible 요구 슬롯이 다 찼나?(109=Latin+common+한국어 단일형 /
   209=시대좌표·원본/사본·욕망·두려움·관계 / 102=생활권·이념·생태)

결정 규칙: 6 green + 수요 있음 → 승격 / 수요 없지만 안정·무충돌 → 선택적 / 충돌·유동·
미충족 → 고친 뒤 / 아무도 안 쓰는데 파급만 큼 → 보류.
