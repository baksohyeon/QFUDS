# QFUDS Verse 세계관 한 장 기준서

문서가 너무 많아 처음 읽는 사람이 방향을 잃는 문제를 줄이기 위한 **이해용 압축

## 한 문장

위조가 완벽해진 미래에서, 무엇이 진짜였는지를 판정하는 권력이 죽음과 기억까지
소유하려 하자, 사람들은 자기 상실을 누가 쓸 권리가 있는지 다툰다.

## 가장 짧은 설명

처음에는 사진, 영상, 목소리, 기록을 믿을 수 없게 된다. 그래서 "이게 진짜인가"를
판정하는 일이 세상에서 가장 강한 권력이 된다. 그 권력은 회사, 검증기관, 법원,
아카이브로 자란다.

그다음에는 암호와 원장의 바닥이 흔들린다. 예전에는 잠긴 것을 주인만 열 수 있다는
믿음이 있었다. 그 믿음이 깨지면 돈, 소유, 신원, 계약, 기록의 진짜성이 흔들린다.

마지막으로 죽음이 상품이 된다. 다만 이 세계의 복원은 부활이 아니다. 남은 데이터로
사람을 재구성해 인공 신체에 올리는 사본이다. 원본은 돌아오지 않는다. 그런데 권력은
그 사본을 "진짜 귀환"으로 팔고, 가족과 시장과 법은 그 사본을 청구하려 한다.

## 현재 시간 척추

하드 연도는 확정하지 않는다. 현재 기준은 [125 Q-Day 타임라인 스파인](10_world/125_qday_two_crisis_timeline_spine_ko.md)이다.

| 층 | 핵심 사건 | 의미 |
| --- | --- | --- |
| 근미래 인식 위기 | 딥페이크, AI, 기록 편집으로 진짜성 붕괴 | 검증경제와 Aletheia Systems가 생긴다 |
| 인식 위기 후기 | 인간 확인 루프와 검증자 도장이 제도화 | 사람이 "예/아니오"를 비준해야 효력이 생긴다 |
| 먼 미래 Q-Day | 암호 서명과 옛 원장의 바닥이 소급 붕괴 | 소유, 신원, 과거 기록의 권위가 흔들린다 |
| 복원 시대 | 죽은 자의 데이터 사본을 인공 신체에 재현 | 복원, 청구권, 망각권, 동일성 논쟁이 열린다 |

## 세계 규칙

| 규칙 | 현재 기준 |
| --- | --- |
| 복원 | 부활이 아니라 데이터 재구성 사본이다. 원본은 돌아오지 않는다. |
| 물리 | 열역학 제2법칙은 깨지지 않는다. 엔트로피 역전은 금지된 신화다. |
| 검증 | 기계는 계산하지만, 법적 효력은 사람이 비준해야 생긴다. |
| 권력 | "진짜"를 정하는 자가 상속, 소유, 기억, 죽음의 의미를 정한다. |
| QFUDS | 이 세계 안에서는 일부 세력이 신화나 금지 이론처럼 사용할 수 있지만, QFUDS 연구 증거가 아니다. |

| 세력 | 역할 |
| --- | --- |

| Laur Observatory | 복원 claim의 정의, 가정, unknown을 감사하는 조직. |

| Aletheia Veil | 복원되지 않을 권리와 삭제된 자의 계승권을 지키려는 내부 이탈 계열. |
| Domus Clavium | key와 원장, 복원 인프라를 소유권으로 묶으려는 장부 가문. |
| 망각권 진영 | 죽을 권리, 잊힐 권리, 복원 거부권을 주장하는 세력들. |
