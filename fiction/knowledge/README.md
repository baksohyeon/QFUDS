# Knowledge / Zettelkasten

작품과 세계 하나에 종속되지 않고 다른 창작에서도 재사용할 수 있는 생각을
둔다. 이 선반이 fiction studio의 Zettelkasten이다. 원자화 큐나 자동 생성 카드의
저장소가 아니다.

한 파일에 한 논지를 둘 수는 있지만, 파일 크기를 작게 유지하는 것이 목표는
아니다. 각 노트는 자기 말로 쓰고 왜 연결되는지 표준 Markdown 링크와 본문으로
설명한다.

## 폴더 역할

- `notes/`: 독립적으로 다시 쓸 수 있는 idea, question, pattern, principle,
  technique, observation
- `maps/`: 여러 Zettel의 관계와 긴장을 직접 배열한 구조 노트
- `seeds/`: 여러 Zettel 사이에 인물·갈등·형식 후보가 생긴 이야기 씨앗

## 원자성 기준

원자적이라는 뜻은 짧다는 뜻이 아니다. 다른 작품에서도 독립적으로 이해하고
발전시킬 수 있는 한 논지인가를 본다. 한 번만 쓰이는 인물의 눈 색, 특정 행성의
달 개수, 7장의 폭발은 각각 world나 project에 둔다.

링크에는 관계를 적는다.

```text
supports | contradicts | extends | causes | exemplifies | transforms
```

링크 수보다 왜 연결되는지 본문에 쓰는 것이 중요하다. Zettel이 작품 후보로 자라면
원본을 옮기거나 복사하지 않고 `seeds/`와 project README가 링크한다.
