# Research anchors

현실의 과학·역사·사회·문화 자료를 픽션의 핍진성에 활용하기 위한 선반이다.
여기 있는 내용은 QFUDS 연구 증거가 아니다. 외부 자료를 새로 다룰 때는
[Fiction Source Intake Workflow](../../.agent/workflows/fiction-source-intake-workflow.md)를
적용한다.

## 구조

- `sources/<source-id>.md`: URL, 검사 상태, 허용 사실, 금지 주장, 창작적 비약
- `assets/<source-id>/`: 라이선스와 필요성이 확인된 로컬 사본만 저장
- 주제별 종합 노트: 여러 source record를 연결하되 출처별 경계를 유지

픽션 자료를 `docs/wiki/research/`에 넣지 않는다. 같은 자료를 QFUDS 연구 근거로
쓰려면 그 용도에 대해 별도 연구 워크플로를 실행한다.
